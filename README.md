# Todo Application
This is a simple Todo application built with Django for managing daily tasks efficiently.

## How to Run Locally
1. Clone the repository: git clone https://github.com/parvesweb/my-todo-assignment.git
2. Install dependencies: pip install -r requirements.txt
3. Run migrations: python manage.py migrate
4. Start server: python manage.py runserver

## Live Deployed URL
You can access the live application here: https://my-todo-assignment.onrender.com/
